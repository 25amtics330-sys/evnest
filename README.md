## ✨ Overview

**EVNest** is a cutting-edge web application designed to connect electric vehicle (EV) owners with charging station hosts. Whether you're looking for a quick charge on a road trip, or you have a charger at home you'd like to share, EVNest provides a seamless, beautiful, and highly interactive platform to make it happen. 🌍 🔋

With a gorgeous user interface featuring a dynamic map, dark mode support, and interactive bookings, it represents the modern standard for eco-friendly marketplace applications.

<br />

## 🚀 Key Features

- 🗺️ **Interactive Charging Map:** Discover available chargers near you with an interactive, responsive map interface.
- 🌙 **Modern UI with Dark Mode:** Beautiful glassmorphism UI, vibrant colors, and seamless dark mode transitions.
- 📅 **Real-time Bookings:** Schedule charging sessions effortlessly with an intuitive booking system.
- 🔐 **Secure Authentication:** User registration, login, and secure session management.
- 🤖 **AI Assistant Integration:** Powered by an intelligent AI feature for an enhanced user experience.
- ⚡ **Lightning Fast:** Built with Vite for instant server starts and lightning-fast HMR.

## 💻 Getting Started Locally

To run EVNest on your local machine, follow these simple steps.

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/evnest.git
cd evnest
```

### 2. Start the Backend
```bash
cd server
npm install
npm run dev
```

### 3. Start the Frontend
Open a new terminal window:
```bash
cd client
npm install
npm run dev
```

Your app should now be running! Frontend on `http://localhost:5173` and Backend on `http://localhost:5000`.

<br />

## 🌐 Deployment Architecture

EVNest is configured for automated, seamless deployment:
- The **`client/`** directory is optimized for deployment on **Vercel**.
- The **`server/`** directory is configured to run instantly on **Render** utilizing the `render.yaml` infrastructure-as-code configuration.
